# Gameverse Frontend

React + Vite alapú kliens a Gameverse projekthez.

## Fejlesztés

```bash
npm install
npm run dev
```

Az alkalmazás fejlesztői módban Vite proxyval a `http://localhost:8080` backendhez csatlakozik.

## Fő funkciók

- játéklista és játékoldalak
- globális és játékonkénti ranglista
- bejelentkezés, regisztráció, jelszó-visszaállítás
- profilkezelés és nyelvváltás
- admin felület felhasználók és játékok kezeléséhez
